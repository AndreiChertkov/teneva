:teneva:
  Python package **teneva** (**ten**\ sor **eva**\ luation) provides a very compact implementation of basic operations in the tensor-train (TT) format, including TT-SVD, TT-ALS, TT-ANOVA, TT-cross, TT-truncate, "add", "mul", "norm", "mean", etc. The program code is organized within a functional paradigm and it is very easy to learn and use.

  Current version "0.8.5".
